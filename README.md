# Immersio - Générateur de Devis PDF

Un outil moderne et interactif pour générer des devis au format PDF pour les prestations d'Immersio (visites virtuelles 3D, plans 2D, photos, vidéos, etc.).

## 🚀 Fonctionnalités

- **Informations Client** : Saisie facile des coordonnées du client.
- **Type de Bien & Superficie** : Calculateur automatique de coefficients basés sur le type et la taille de l'espace.
- **Prestations & Options** : Sélection dynamique des prestations additionnelles (Plan 2D, Vidéo HD, Photos HD, Tags interactifs, Branding personnalisé, Publication Google Maps).
- **Hébergement** : Options d'hébergement flexibles avec coefficients automatiques.
- **Calculateur Automatique** : Système de tarification intelligent basé sur des multiplicateurs avec possibilité de forcer manuellement les tarifs ou de réinitialiser.
- **Génération PDF** : Export instantané et esthétique du devis en PDF via `jspdf`.

## 🛠️ Stack Technique

- **Structure** : HTML5
- **Style** : CSS3 moderne (Vanilla) avec design premium sombre et responsive.
- **Logique** : JavaScript (Vanilla ES6)
- **Librairies** : [jsPDF](https://github.com/parallax/jsPDF) pour la génération de PDF côté client.

## 📦 Installation et Utilisation

1. Clonez ce dépôt.
2. Ouvrez le fichier `pdf_creator.html` directement dans votre navigateur web préféré.
3. Remplissez les champs et cliquez sur **Générer le PDF** !
